How to set time
Write a number on the canvas. Please check each tab links to check how to write each number on a unistroke with animation gifs.
Clear time
Write a 'c' anywhere on the timer. write
Start timer
Make a single click anywhere on the timer.
Reset timer
Make a double click anywhere on the timer.
PC/Tablet/Smartphone
If you wanna use table or smartphone, please make sure that your browser is not in desktop mode. On smartphone/table, WritableTimer is supposed to be used portrait mode because of noscrool mode.

How to use jsWritableTimer on OBS
  1. Copy share link by clicking copy share link button on the page.
  2. Open OBS and add browser source and set belows
    • URL: Paste your share link
    • Width: 1280
    • Height: 720
    • *The canvas size is fixed on 16:9 aspect ratio.
  3. Set your timer on Interactive mode of browser source.
